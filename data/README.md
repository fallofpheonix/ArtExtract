# Data Directory

This directory stores small, versioned data assets required to run the repository.

## Intended Contents

1. manifests used by scripts and tests
2. metadata splits and class maps
3. provenance templates

## Do Not Store Here

1. large raw image archives
2. non-redistributable source images
3. generated experiment outputs
4. one-off notebook exports

## Current Subdirectories

1. `manifests/`
2. `metadata/`
3. `templates/`

Dataset rights and release rules are defined in [../docs/DATA_POLICY.md](../docs/DATA_POLICY.md) and [../DATA_RIGHTS.md](../DATA_RIGHTS.md).
