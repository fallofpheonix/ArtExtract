# Notebooks

Only source notebooks belong in this directory.

## Rules

1. keep `.ipynb` source notebooks that capture reproducible analysis or submission material
2. do not keep executed copies, HTML exports, PDF exports, or ad hoc images unless they are explicitly required for a submission bundle
3. if a notebook becomes operationally important, move the logic into `scripts/` or `src/artextract/`
